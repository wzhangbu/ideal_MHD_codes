# ideal_MHD_codes

Use IE_Figure to generate the convection maps in the ionosphere from IE output in SWMF model 

Use 9.5RE_keogram to generate the U_pole at open-closed boundaries in the magnetosphere.

Use the MP_keogram_tangential to get the magnetopause plots and parameters.

Use the MP_Bz_-5 to get the magnetosheath upstream information.
